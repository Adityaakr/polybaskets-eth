ETF-style prediction-market baskets on Vara.eth (Ethereum-settled / ethexe). Bundle Polymarket
outcomes into a weighted basket and bet on it as a single on-chain position — gasless gameplay
via injected pre-confirmations, ETH + wVARA collateral, embedded-wallet (Privy) login.
